# 技巧积累

> 辗转相除法

求两个数的最大公约数

```java
private int gcd(int a, int b){
       while(b != 0){
           int tmp = b;
           b = a%b;
           a = tmp;
       }
       return a;
   }
```

> 整数除法保留两位小数

利用 String.format方法

```java
        int a = Integer.valueOf("13".substring(0, 2));
        double b = a - a * ((double)50 / 100);
        String c = String.format("%.2f",b);
        System.out.println(c);
```
